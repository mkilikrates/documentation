# Enabling Kubernetes on Docker Desktop with kubeadm

You can follow the [official documentation](https://docs.docker.com/desktop/use-desktop/kubernetes/) on how to enable, basic steps are:
- [Install docker desktop](https://docs.docker.com/get-started/get-docker/)
- Open docker desktop
- Go to settings
- Go to Kubernetes
- Enable Kubernetes and select option kind, the version of your cluster and number of nodes
- Apply will restart docker desktop

# Disable Kubernetes

- Go to settings
- Go to Kubernetes
- Disable Kubernetes

# Reset Kubernetes

- Go to settings
- Go to Kubernetes
- Reset Kubernetes
